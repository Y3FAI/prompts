# LLM & Agent Prompt Collection

A personal collection of prompts for Large Language Models and AI agents. 

## About

This repository contains prompts I've created or found and curated for working with various LLMs and AI agents. The prompts are organized by use case and optimized for clarity and effectiveness.

## Structure

```
prompts/
├── general/          # General-purpose prompts
├── writing/          # Content creation and writing prompts
└── agents/           # Agent-specific system prompts
```

## Usage

1. Browse the collection to find prompts relevant to your task
2. Copy the prompt and customize it for your specific needs
3. Test and iterate based on your LLM's responses

## Tips

- Adjust the temperature and other parameters based on your use case
- Add specific context and examples for better results
- Iterate on prompts based on output quality

This is a personal collection, but feel free to fork and adapt for your own use.
